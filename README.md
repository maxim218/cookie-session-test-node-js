# Сессии NodeJs

Установка библиотек

```
npm install
```

Запуск приложения

```
npm start
```

## API

Выход из системы

```
http://localhost:5000/exit
```

Вход в систему

```
http://localhost:5000/man?man=george
```

Увеличение счётчика посещений

```
http://localhost:5000/number
```
